WordPress Plugins

I enjoy developing plugins for WordPress and I love working on new ideas. Here is a list of plugins I’ve released so far.

  • Code Snippets
  • Dev Profiles
  • Enable CodePress
  • Minimum Password Length
  • Minimum Password Strength
  • Naughty Monkey
  • Random Headers
  • Remove Editor/Install Pages
  • Short Comment Filter
  • WP SendGrid

I’ve had fun developing these plugins and have found them useful, so I want to share them with you. Using them places you under no obligation, but if you’d like to say thanks, there are a couple ways you can do so.

1. Just Say Thanks Are you using one of my plugins? I’d love to know what you think. Have an idea for an improvement, or want to report a bug? You can grab me at will@itsananderson.com and let me know.

2. Hire Me If you’re working on a project that needs custom Theme/Plugin work, I’m available for hire. I’m not always able to take on new projects, but if you’re interested in working with me, send me an email describing what you want and I’ll let you know.

3. Donate If you want to send a “monetary thank you note” I won’t say no to PayPal donations. You can send a donation directly to will@itsananderson.com, or just click the shiny little Donate button below.
